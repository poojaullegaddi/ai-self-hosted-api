# 🚀 Self-Hosted AI API (FastAPI + Ollama)

## 📌 Overview
This project implements a self-hosted AI API using FastAPI and Ollama.  
It allows users to send prompts via HTTP requests and receive responses generated by a locally hosted LLM (Mistral).

Unlike external APIs, this solution ensures:
- 🔒 Data privacy (no external API calls)
- 💰 Zero API cost
- ⚙️ Full control over model and deployment

---

## 🧠 Architecture

User → FastAPI → Ollama → Mistral Model → Response → User

---

## 🛠️ Tech Stack

- Python
- FastAPI
- Ollama (Local LLM runtime)
- Mistral Model
- Uvicorn
