# 🚀 Self-Hosted AI API (FastAPI + Ollama)

## 📌 Overview
This project implements a self-hosted AI API using FastAPI and Ollama.  
It allows users to send prompts via HTTP requests and receive responses generated by a locally hosted LLM (Mistral).

Unlike external APIs, this solution ensures:
- 🔒 Data privacy (no external API calls)
- 💰 Zero API cost
- ⚙️ Full control over model and deployment

---

## 🧠 Architecture

User → FastAPI → Ollama → Mistral Model → Response → User

---

## 🛠️ Tech Stack

- Python
- FastAPI
- Ollama (Local LLM runtime)
- Mistral Model
- Uvicorn

---
⚙️ Setup Instructions
1️⃣ Install dependencies
pip install -r requirements.txt

2️⃣ Install and run Ollama

Download from: https://ollama.com
---
Run model:

ollama run mistral
---
⚠️ Keep this terminal running

3️⃣ Run API server
uvicorn app:app --reload
4️⃣ Access API Docs
---
Open in browser:

http://127.0.0.1:8000/docs
---
🧪 API Usage
Endpoint
POST /generate

Headers
api_key: mysecret123
---

Request Body
{
  "prompt": "What is API?"
}

---

Response
{
  "response": "API is a set of rules..."
}
